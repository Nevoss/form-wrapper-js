Form Wrapper JS

A lightweight library that creates forms systems in a convenient and easy way, without dependencies and magic code.

Get Started →

No design decision

Create your own html and styles, the library just handle the form values and meta data.

Customizable

You can hook up to the library methods and customize the form via options, with that you can chose how your form will behave.

Easy to use and to extend

You can start with a simple form, and later you can extend the form and make it very flexible and powerful.
